# 🗺 Roadmap проекта DigitalCoreAnalyser

> **Актуальность:** Июнь 2026  
> **Автор:** Горшков Артём  
> **Репозиторий:** [DigitalCoreAnalyser](https://github.com/4y8a4ek/DigitalCoreAnalyser)

Проект находится в стадии активной разработки. Ниже представлен план развития на ближайшие месяцы.

---

## ✅ Завершено (v1.0 - Июнь 2026)

- [x] Базовый функционал приложения на C# / WinUI 3 / .NET 8
- [x] Архитектура на паттерне MVVM
- [x] Загрузка бинарных RAW-файлов (воксельные данные)
- [x] 2D-визуализация срезов (XY, XZ, YZ) с навигацией по слоям
- [x] 3D-визуализация порового пространства (облако точек, вращение, масштабирование)
- [x] Расчёт общей пористости
- [x] Выделение связного кластера (алгоритм BFS)
- [x] Расчёт эффективной пористости
- [x] Алгоритм Козени–Кармана (проницаемость)
- [x] Формула Арчи (фактор формации)
- [x] Анализ подвыборок (RVE / SVE)
- [x] Защита ВКР и апробация на образцах Berea и S7

## 🔬 Долгосрочная цель (2027+)

> **Переход к математическому моделированию многофазной фильтрации** на основе цифровых моделей керна. Это позволит прогнозировать поведение жидкостей в реальных пластовых условиях без физических экспериментов и выведет проект на новый научный уровень.
